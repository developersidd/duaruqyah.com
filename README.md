# 📿 Dua Ruqya – Islamic Duas & Azkar Platform

A web-based platform for accessing daily Islamic duas and azkar, organized by categories for easy navigation. Built with modern web technologies, it provides a clean, structured, and user-friendly reading experience.

---

## 🚀 Features

- 📚 Categorized collection of duas and azkar  
- 🧭 Sidebar-based navigation for quick access  
- 📖 Custom dua detail page for focused reading  
- 🖥️ Dashboard layout (categories, content, settings)  
- 🌐 Responsive design for all devices  
- ⚡ Fast frontend powered by Next.js  
- 🗄️ PostgreSQL database (Neon) for scalability  

---

## 🧰 Tech Stack

- **Frontend:** Next.js, TypeScript  
- **Backend:** Express.js  
- **Database:** PostgreSQL (Neon)
